bootstrap2-columns-under768
===========================

Bootstrap 2: Columns under 768px

Set of extra CSS rules to create columns under 768px using bootstrap 2

|   column 1   |   <- tablets, mobile
|   column 2   |
|   c1  |  c2  |
| c1 | c2 | c3 |
